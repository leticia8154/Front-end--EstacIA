# Front-end--EstacIA
## EstacIA - Sistema de Gerenciamento de Estacionamento Inteligente


O *EstacIA* é uma solução mobile voltada para o gerenciamento, monitoramento e otimização de estacionamentos em centros comerciais e shopping centers.A plataforma funciona como uma camada de inteligência digital aplicada à infraestrutura existente, eliminando a necessidade de sensores eletrônicos caros por meio do uso estratégico de QR Codes e localização do carro.

## 🚀 Funcionalidades Principais (MVP)

- *Seleção de Destino:* Escolha do shopping antes de iniciar a viagem.
- *Navegação Inteligente:* Rota integrada via GPS até a vaga disponivel e função reversa "Leve-me ao meu carro".
- *Check-in Descomplicado:* Registro de posição instantâneo via leitura de QR Code utilizando a placa ou chassi do veículo.
- *Monitoramento e Alertas:* Notificações em tempo real sobre o tempo de permanência e tarifas acumuladas.
- *Checkout Digital:* Pagamento integrado e seguro via Pix ou Cartão de Crédito com janela de tolerância de 20 minutos para saída.
- *Painel Administrativo:* Dashboard com relatórios de uso, rotatividade de vagas e faturamento para os gestores do shopping.

## 🛠️ Tecnologias Utilizadas

- *Design/Prototipagem:* Figma
- *Front-end:* HTML5, CSS3, JavaScript (Estruturação de telas no VS Code)
- *Modelagem:* Engenharia de Software / UML (Diagramas de Caso de Uso e Classes)
- *Metodologia:* Scrum / Kanban
